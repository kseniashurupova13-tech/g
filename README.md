# Book Tracker — Приложение для учета прочитанных книг
**Автор:** [Шурупова Ксения]

## Описание программы
Программа предназначена для ведения личного списка прочитанных книг. Пользователь может сохранять название, автора, жанр и объем книги. Все данные автоматически сохраняются в формате JSON и доступны после перезагрузки.

## Функционал
- **Добавление данных:** Форма для ввода информации о новой книге.
- **Валидация:** Проверка на пустые поля и контроль ввода числовых данных (страницы).
- **Фильтрация:** Поиск книг по жанру или по количеству страниц (показ книг объемом более указанного).
- **Хранение:** База данных в файле `books_data.json`.

## Инструкция по запуску
1. Установите Python 3.x.
2. Скачайте файл `main.py`.
3. Запустите приложение командой: `python main.py`.

## Примеры тестирования
1. **Тест добавления:** Введите "Капитанская дочка", "Пушкин", "Классика", "200". Книга должна появиться в таблице.
2. **Тест валидации:** Оставьте поле "Автор" пустым и нажмите "Добавить" — программа выдаст предупреждение.
3. **Тест фильтрации:** Добавьте книгу со 100 страницами и книгу с 300 страницами. Введите в фильтр страниц "200" — в таблице останется только книга на 300 страниц.
